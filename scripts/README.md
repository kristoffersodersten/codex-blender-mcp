# Scripts

Scripts in this directory are development and migration helpers.

Supported public API:

- MCP tools in `src/`
- measurement contracts in `src/measurementContracts.ts`
- Blender bridge operations in `blender/bridge.py`

Everything else in this directory should be treated as experimental unless a script has its own usage contract and test coverage.

Do not use experimental scripts as the product export source of truth. Permit-support exports must come from locked Blender orthographic views through the MCP workflow.
